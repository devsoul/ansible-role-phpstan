PHPStan Ansible role
=========

If you are a PHP developer and you use PHPStan.

Requirements
------------
  - `php > 7.1` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.phpstan

Check if it's installed:
------
    phpstan --version
